\chapter*{서문}
\addcontentsline{toc}{chapter}{서문}

이 책은 Hugging Face로 텍스트 분석을 시작하려는 실무자를 위한 실습형 원고입니다. 특히 ML, BERT, LLM의 변화를 너무 빠르게 지나오느라 한 번쯤 전체 흐름을 정리하고 싶은 Data Scientist, Data Engineer, AI Scientist, AI Engineer를 독자로 생각했습니다.

시작점은 BERT가 아닙니다. 먼저 훨씬 작고 투명한 도구인 \api{TfidfVectorizer}와 \api{LogisticRegression}에서 출발합니다. 텍스트가 숫자가 되고, 숫자가 출력 헤드를 지나고, 출력이 Loss로 평가되는 흐름을 먼저 손에 익히기 위해서입니다.

현재 원고는 Phase 0과 Phase 1, 1--14장까지를 담습니다. 각 장은 Google Colab 노트북을 바탕으로 만들었습니다. 노트북의 실행 흐름은 유지하되, 책에서는 각 장에서 무엇이 바뀌는지, 코드가 어떤 의미를 갖는지, 출력 결과를 어떻게 읽어야 하는지 더 차분히 따라갈 수 있도록 설명을 보강했습니다.

읽을 때는 각 장의 변화추적표와 Loss 설명을 먼저 훑어본 뒤 코드를 따라가면 좋습니다. 그러면 “이 장에서 정확히 무엇이 바뀌었는지”가 더 선명하게 보입니다.
